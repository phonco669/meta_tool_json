# JSON元数据生成工具 - 项目说明文档

## 1. 项目规划

开发一个基于React + Vite + Tailwind CSS的JSON元数据生成工具，实现文本结构与JSON格式的双向实时转换功能。

**核心功能：**
- 左侧窗口编辑层级文本，右侧实时渲染JSON格式
- 支持反向操作：粘贴JSON自动生成结构文本
- 一键复制功能，支持JSON和文本内容的快速复制
- Windows平台下一键运行的桌面应用

**目标用户：**
- 开发人员需要快速生成JSON配置文件
- 数据分析师处理结构化数据
- 任何需要在文本结构和JSON之间转换的用户

## 2. 实施方案

### 阶段一：项目初始化
- 使用Vite创建React项目
- 配置Tailwind CSS样式框架
- 设置项目基础结构和依赖

### 阶段二：UI界面实现
- 实现左右分栏布局（Split Pane）
- 左侧：可编辑的文本区域，支持层级缩进
- 右侧：JSON预览区域，支持语法高亮
- 添加复制按钮和状态提示

### 阶段三：核心转换逻辑
- 实现文本到JSON的解析器
  - 支持缩进表示层级关系
  - 处理基本数据类型（字符串、数字、布尔值）
  - 支持数组和嵌套对象结构
- 实现JSON到文本的反向转换
  - 保持原有层级结构
  - 格式化输出，便于编辑

### 阶段四：增强功能
- 添加错误处理和验证
- 实现实时同步更新
- 添加导入/导出功能
- 优化用户体验和界面交互

### 阶段五：打包发布
- 配置Electron打包Windows桌面应用
- 优化构建配置和性能
- 生成可执行安装程序
